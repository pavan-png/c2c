# C2C React Website

A premium React + Vite + Tailwind CSS website built from the original static HTML structure, redesigned with:

- Modular industry-style folder structure
- Reusable section components
- Framer Motion transitions and section reveals
- Animated sticky navigation
- Responsive layout
- Cleaner production-ready code organization

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- The overall content structure is preserved from the original HTML.
- The founder image currently uses a remote placeholder. Replace it in `src/data/siteData.js` if needed.
- The `Careers` menu smoothly navigates to the Placement section, matching the original structure.
